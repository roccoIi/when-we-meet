## 필요한 스크린샷 목록

### 1. login.png
- **페이지**: 로그인 페이지
- **내용**: Google, Kakao 로그인 버튼이 보이는 화면
- **권장 크기**: 1200x800px

### 2. create-meeting.png
- **페이지**: 미팅 생성 페이지
- **내용**: 미팅 제목, 설명, 날짜 범위를 입력하는 폼
- **권장 크기**: 1200x800px

### 3. meeting-list.png
- **페이지**: 미팅 목록 페이지
- **내용**: 생성/참여 중인 미팅 카드들이 나열된 화면
- **권장 크기**: 1200x800px

### 4. schedule-input.png
- **페이지**: 스케줄 입력 페이지
- **내용**: 캘린더에서 시간을 선택하는 화면
- **권장 크기**: 1200x800px

### 5. meeting-detail.png
- **페이지**: 미팅 상세 페이지
- **내용**: 참여자들의 스케줄이 종합된 화면 (겹치는 시간대 강조)
- **권장 크기**: 1200x800px

### 6. invite-link.png
- **페이지**: 초대 링크 공유
- **내용**: 초대 링크를 복사하는 모달이나 버튼이 보이는 화면
- **권장 크기**: 800x600px

## 스크린샷 촬영 팁

1. **브라우저**: Chrome 개발자 도구의 디바이스 툴바로 일관된 크기 설정
2. **해상도**: Retina 디스플레이 고려하여 2x 크기로 촬영 후 50% 축소
3. **내용**: 실제 데이터를 사용하되 개인정보는 마스킹
4. **형식**: PNG (투명 배경 필요시) 또는 JPG (일반적인 경우)
5. **파일명**: 위에 명시된 이름 그대로 사용
